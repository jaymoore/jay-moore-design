Round 3 · pre-step · type specimen lock

Four candidate type pairs. Grayscale. No layout, no color.

Codex R7's keystone fix: lock type before palette. Each card sets the same three specimens that matter for this site — the heading (pivot statement), the paragraph (demo caption / about prose), and the code/label block (the mono that carries the instrument rigor). Same content everywhere, so the only variable is the letterforms.

How to read this: the structure (Iteration 3) earned its instrument identity in grayscale. The type has to hold that identity on letterforms alone.
  1 — heading: confident hand without shouting? a real point of view, not Geist-default, not invisible?
  2 — paragraph: calm and legible at a 50ch+ measure? does it want to be read?
  3 — code/label: does the mono read as structural — an instrument — or as a decorative "techy" font?
Exit: lock ONE pair. Then three palette treatments render on it.
